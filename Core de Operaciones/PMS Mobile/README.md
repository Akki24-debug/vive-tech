# PMS Mobile

App movil interna del PMS para iOS y Android.

## Variables de entorno

- `EXPO_PUBLIC_PMS_MOBILE_API_URL`
- `EXPO_PUBLIC_PMS_MOBILE_TENANT_ID`
- `EXPO_PUBLIC_PMS_MOBILE_COMPANY_CODE`
- `EXPO_PUBLIC_PMS_MOBILE_USER_ID`
- `EXPO_PUBLIC_PMS_MOBILE_ACTOR_USER_ID`

## Comandos

- `npm install`
- `npm run start`
- `npm run android`
- `npm run ios`
- `npm run typecheck`
